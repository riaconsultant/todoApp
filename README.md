# ToDo Application 
todoApp in Angularjs. 

1. Add new ToDo task.
2. Edit selected ToDo task.
3. Delete selected ToDo task.
4. Show All ToDo List task, Mark completed task.
5. Show Only Pending tasks.
6. Show Only completed tasks.

# Technology
1. Angularjs
2. Nodejs
3. Express
4. MongoDB
